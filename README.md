# GitLab_Day02

To remove branch locally
```
git branch -d branchName
```

To remove branch remotly
```
git branch -D branchName
```
To list tags
```
git tag
```

To remove tag locally
```
git tag -d tageName
```

To remove tag remotly
```
git push --delete origin tageName
```
